# SaaS-Project: Agent Command Center 🤙

## Concept
Dashboard for monitoring and controlling multiple AI agents.

## Core Features
1. **Agent Cards:** Status, Model name, Current Task, Performance metrics.
2. **Global Logs:** Combined stream of all agent outputs.
3. **Action Center:** Buttons to "Kill", "Restart" or "Steer" specific agents.
4. **Tech Overview:** Visualized tech stack (Next.js/TS).

## Tech Stack
- Frontend: Next.js + Tailwind.
- Icons: Lucide (for agent roles).
- Data source: Local `.openclaw` status & logs.
