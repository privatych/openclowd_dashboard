## 🏢 Overview
**ACC CORE** is a mission control center for your AI agent workforce. Instead of dry terminal logs, you manage your agents in a **top-down Pixel-Art Office** emulator.

## 🕹️ Features
- **Live Office Sync:** Visual representation of agent status (Online, Busy, Offline).
- **RPG Metrics:** Each agent has **Integrity (HP)** and **Resources (MP)** levels.
- **System HUD:** Real-time monitoring of Node CPU, RAM, and Daily Spends ($).
- **Interactive Bubbles:** Hover over agents to see their current task, model (Gemini/Claude/GPT4), and level.
- **Global Event Log:** Matrix-style scrolling terminal in the footer.
